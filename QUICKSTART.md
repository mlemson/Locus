# Quick Start Guide - LOCUS

## Voor nieuwe gebruikers / For new users

### Nederlands

**Snelste manier om te beginnen:**

1. Download de bestanden:
   ```bash
   git clone https://github.com/mlemson/Locus.git
   cd Locus
   ```

2. Open in VS Code:
   ```bash
   code .
   ```

3. Installeer Live Server extensie in VS Code

4. Open `index.html` en klik op "Go Live"

**Of zonder Git:**
- Download ZIP van GitHub → Uitpakken → Open map in VS Code

---

### English

**Fastest way to get started:**

1. Download the files:
   ```bash
   git clone https://github.com/mlemson/Locus.git
   cd Locus
   ```

2. Open in VS Code:
   ```bash
   code .
   ```

3. Install Live Server extension in VS Code

4. Open `index.html` and click "Go Live"

**Or without Git:**
- Download ZIP from GitHub → Extract → Open folder in VS Code

## Commands Cheat Sheet

### Git Commands
```bash
# Clone repository
git clone https://github.com/mlemson/Locus.git

# Get latest changes
git pull origin main

# Check status
git status

# Commit changes
git add .
git commit -m "Your message"
git push origin main
```

### VS Code Shortcuts
- `Ctrl/Cmd + P` - Quick file open
- `Ctrl/Cmd + Shift + F` - Search in files
- `Ctrl/Cmd + B` - Toggle sidebar
- `Alt + Shift + F` - Format code

## Files Overview

- `index.html` - Main game (open this to play)
- `editor.html` - Level editor (open this to create levels)
- `editor.js` - Editor logic
- `editor.css` - Editor styles
